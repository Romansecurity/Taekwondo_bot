# 🤸 Бот записи на секцию

Telegram-бот для записи детей на спортивные секции. Позволяет выбрать район города, посмотреть тренеров и оставить заявку на первую тренировку.

---

## 📋 Возможности

- Выбор района из 13 доступных локаций
- Просмотр списка тренеров по выбранному району
- Анкета тренера с фото, телефоном и адресами залов
- Запись на занятие через пошаговую форму (имя родителя, телефон, имя и возраст ребёнка)
- Сохранение заявок в базу данных
- Административная панель (добавление/удаление тренеров)

---

## 🗂 Структура проекта

```
project/
├── main.py                    # точка входа, регистрация роутеров
├── config.py                  # токен бота (TOKEN)
├── photos/                    # фото тренеров (формат: {trainer_id}.JPG)
│
├── routers/
│   ├── start.py               # команда /start
│   ├── admin.py               # добавление тренеров (админка)
│   └── admin_del_db.py        # удаление записей из БД (админка)
│
├── handlers_fsm/
│   └── callbacks.py           # основная логика: выбор района/тренера, запись
│
├── keyboards/
│   └── inline.py              # inline-клавиатуры (районы, кнопки навигации)
│
└── data_base/
    └── models.py              # работа с БД: создание таблиц, запросы, сохранение
```

## ⚙️ Установка и запуск

### 1. Клонировать репозиторий

```bash
git clone https://github.com/ВАШ_USERNAME/ВАШ_РЕПОЗИТОРИЙ.git
cd ВАШ_РЕПОЗИТОРИЙ
```

### 2. Создать виртуальное окружение и установить зависимости

```bash
python -m venv venv
source venv/bin/activate       # Linux/macOS
venv\Scripts\activate          # Windows

pip install -r requirements.txt
```

### 3. Настроить переменные окружения

Создайте файл `config.py` в корне проекта:

```python
TOKEN = "ВАШ_TELEGRAM_BOT_TOKEN"
```

> Получить токен можно у [@BotFather](https://t.me/BotFather)

### 4. Запустить бота

```bash
python main.py
```

## 🗄 База данных

При первом запуске автоматически создаются таблицы:

- **Тренеры** — имя, телефон, район, адрес зала
- **Заявки** — имя родителя, телефон, имя и возраст ребёнка, выбранный тренер

## 🔄 Сценарий пользователя

```
/start
  └── Выбрать зал
        └── Выбор района
              └── Список тренеров
                    └── Анкета тренера (фото + телефон + адреса)
                          └── Записаться
                                ├── Имя и фамилия родителя
                                ├── Номер телефона
                                ├── Имя ребёнка
                                └── Возраст ребёнка → ✅ Заявка сохранена
```

## 📁 Фото тренеров

Фотографии хранятся в папке `photos/` и называются по `id` тренера в базе данных:

```
photos/
├── 1.JPG
├── 2.JPG
└── ...
```

Если фото не найдено — бот отображает текстовую анкету и предлагает записаться без фото.
